Fleet Server has been deployed!

To verify the deployment:

  kubectl get pods -n {{ .Release.Namespace }} -l app=fleet-server

To check Fleet Server status:

  kubectl logs -n {{ .Release.Namespace }} -l app=fleet-server -f

To access Fleet Server API:

  kubectl port-forward -n {{ .Release.Namespace }} svc/fleet-server 8220:8220

  # Test the API
  curl http://localhost:8220/api/status
